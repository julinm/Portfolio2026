RECORDIS

Java Android Studio Firebase (Firestore, Storage, Auth) Google Maps SDK for Android Geocoder Glide Git & GitHub
Encuéntralo en GitHub

Recordis es una aplicación Android diseñada para almacenar y revivir recuerdos personales en un mapa interactivo. Los usuarios pueden agregar detalles como título, descripción, imagen, fecha, ubicación y etiquetas, creando un “mapa de vida” personal. La aplicación cuenta con una vista de mapa con marcadores interactivos y una vista de lista con opciones de búsqueda y filtrado. Cada usuario tiene una cuenta privada donde sus recuerdos se almacenan de manera segura.

Funciones principales:

  • Crear, editar y eliminar recuerdos.
  • Mapa interactivo con marcadores y búsqueda.
  • Vista de lista con búsqueda y filtros (etiquetas, fecha, favoritos, palabras clave).
  • Selección de favoritos y subida de imágenes.
  • Autenticación de usuario y almacenamiento privado.

Tecnologías y herramientas: Java, Android Studio, Firebase (Firestore, Storage, Authentication), Google Maps SDK, Glide, arquitectura MVVM, Git/GitHub

El proyecto está completamente funcional, con una arquitectura modular utilizando Activities, Fragments y el patrón Repository. Ya he identificado áreas de mejora y estoy trabajando activamente en optimizar la arquitectura, la usabilidad, el diseño visual y en añadir nuevas funcionalidades como copia de seguridad, recuperación de contraseña, inicio de sesión con Google, creación de etiquetas personalizadas y modo oscuro.

CUENTA BANCARIA - Proyecto durante la formación

Java JavaDoc Conceptos de POO
Encuéntralo en GitHub

Este proyecto es una aplicación educativa de consola en Java desarrollada como parte de mi formación. Simula operaciones básicas de una cuenta bancaria, incluyendo depósitos y retiros, utilizando la clase CCuenta para representar la cuenta y la clase Main para demostrar su funcionamiento. El proyecto hace énfasis en la refactorización, la encapsulación y la documentación adecuada con JavaDoc. Los atributos de la cuenta son privados y se accede a ellos mediante getters y setters, asegurando buenas prácticas de programación orientada a objetos.

El proyecto también incluye integración con Git, con un repositorio público y seguimiento del historial de versiones. El método operativa_cuenta realiza todas las operaciones de la cuenta, gestionando excepciones para transacciones inválidas. Los usuarios pueden compilar y ejecutar el proyecto en cualquier IDE de Java para ver el saldo, realizar depósitos y procesar retiros. En general, la asignatura resalta la importancia del código limpio, la mantenibilidad y la documentación completa.

SISTEMA DE RESERVAS - Proyecto durante la formación

Java Componente Swing
Encuéntralo en GitHub

Este proyecto es una aplicación de escritorio educativa en Java desarrollada como parte de mi formación. Simula un sistema de reservas para el "Salón Habana", permitiendo a los usuarios ingresar detalles de la reserva, como nombre, información de contacto, tipo de evento, número de asistentes y tipo de cocina. Para conferencias, se habilitan dinámicamente opciones adicionales como el número de sesiones y la necesidad de habitaciones. La interfaz incluye validaciones para asegurar que todos los campos requeridos se completen correctamente y proporciona mensajes de confirmación al realizar reservas exitosas.

El proyecto demuestra el uso de componentes Swing, manejo de eventos con ActionListener y ItemListener, y validación de la interfaz con JOptionPane. Los componentes están organizados con paneles, etiquetas, separadores y fuentes personalizadas para mayor claridad y usabilidad. Los usuarios pueden reiniciar el formulario para nuevas reservas, y todos los datos se manejan en memoria. En general, la asignatura resalta el diseño limpio de la interfaz, formularios interactivos y buenas prácticas de validación de entradas.

Chatbot simple

PHP JavaScript HTML CSS Bootstrap MySQL
Encuéntralo en GitHub

JuliBot es un chatbot ligero, interactivo y basado en la web, construido con PHP, MySQL y jQuery. Funciona como un asistente personal, respondiendo preguntas sobre mi trayectoria profesional, incluyendo experiencia laboral, formación académica y tecnologías utilizadas. El chatbot permite interacción en tiempo real mediante AJAX, muestra un historial dinámico de mensajes desde una base de datos MySQL y cuenta con un diseño responsive usando Bootstrap para distintos tamaños de pantalla. La interfaz incluye avatares personalizados, mensajes con marca de tiempo y un diseño limpio y profesional.

El proyecto demuestra habilidades de desarrollo full-stack, incluyendo trabajo frontend con HTML5, CSS3, JavaScript (jQuery) y Bootstrap, así como funcionalidad backend utilizando PHP y MySQL. Los usuarios pueden cambiar entre inglés y español, escribir preguntas y recibir respuestas instantáneas. La base de código incluye archivos modulares para la interfaz, procesamiento de mensajes, conexión a la base de datos y scripts personalizados para el manejo de mensajes. En general, JuliBot resalta aplicaciones web interactivas, manejo dinámico de contenido e interfaces bilingües para el usuario.
