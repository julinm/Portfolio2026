Last project

OCT - DEC 2025

Recordis – Android Application (Final Course Project)

Find it on GitHub

Recordis is an Android app designed to store and revisit personal memories on an interactive map. Users can add details such as title, description, image, date, location, and tags, creating a personal “map of life.” The app features Map View with interactive markers and List View with search and filtering options. Each user has a private account where memories are securely stored.

Main Features:

  • Create, edit, and delete memories.
  • Interactive map with markers and search.
  • List view with search and filters (tags, date, favorites, keywords).
  • Favorites selection and image upload.
  • User authentication and private storage.

Technologies & Tools: Java, Android Studio, Firebase (Firestore, Storage, Authentication), Google Maps SDK, Glide, MVVM architecture, Git/GitHub

The project is fully functional, with a modular architecture using Activities, Fragments, and the Repository pattern. I have already identified areas for improvement and am actively working on enhancing architecture, usability, visual design, and adding new features such as backup functionality, password recovery, Google Sign-In, custom tags, and dark mode.
