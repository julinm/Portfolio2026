RECORDIS

Java Android Studio Firebase (Firestore, Storage, Auth) Google Maps SDK for Android Geocoder Glide Git & GitHub
Find it on GitHub

Recordis is an Android app designed to store and revisit personal memories on an interactive map. Users can add details such as title, description, image, date, location, and tags, creating a personal “map of life.” The app features Map View with interactive markers and List View with search and filtering options. Each user has a private account where memories are securely stored.

Main Features:

  • Create, edit, and delete memories.
  • Interactive map with markers and search.
  • List view with search and filters (tags, date, favorites, keywords).
  • Favorites selection and image upload.
  • User authentication and private storage.

Technologies & Tools: Java, Android Studio, Firebase (Firestore, Storage, Authentication), Google Maps SDK, Glide, MVVM architecture, Git/GitHub

The project is fully functional, with a modular architecture using Activities, Fragments, and the Repository pattern. I have already identified areas for improvement and am actively working on enhancing architecture, usability, visual design, and adding new features such as backup functionality, password recovery, Google Sign-In, custom tags, and dark mode.

BANK ACCOUNT - Educational assignment

Java JavaDoc OOP concepts
Find it on GitHub

This project is an educational Java console application developed as part of my training. It simulates basic bank account operations, including deposits and withdrawals, using a CCuenta class to represent the account and a Main class to demonstrate its functionality. The project emphasizes refactoring, encapsulation, and proper JavaDoc documentation. Attributes of the account are private and accessed through getters and setters, ensuring good object-oriented practices.

The project also includes Git integration, with a public repository and version history tracking. The operativa_cuenta method performs all account operations, handling exceptions for invalid transactions. Users can compile and run the project in any Java IDE to see the account balance, make deposits, and process withdrawals. Overall, the assignment highlights clean code, maintainability, and thorough documentation.

RESERVATION SYSTEM - Educational assignment

Java Swing components
Find it on GitHub

This project is an educational Java desktop application developed as part of my training. It simulates a reservation system for "Salón Habana," allowing users to enter reservation details such as name, contact information, event type, number of attendees, and type of cuisine. For conferences, additional options like the number of sessions and room requirements are enabled dynamically. The interface includes validations to ensure all required fields are completed correctly and provides confirmation messages upon successful reservations.

The project demonstrates the use of Swing components, event handling with ActionListener and ItemListener, and UI validation with JOptionPane. Components are organized with panels, labels, separators, and customized fonts for clarity and usability. Users can reset the form for new reservations, and all data is handled in-memory. Overall, the assignment highlights clean UI design, interactive forms, and proper input validation practices.

Simple Chatbot

PHP JavaScript HTML CSS Bootstrap MySQL
Find it on GitHub

JuliBot is a lightweight, interactive web-based chatbot built with PHP, MySQL, and jQuery. It functions as a personal assistant, answering questions about my professional background, including work experience, education, and technologies used. The chatbot supports real-time interaction using AJAX, displays dynamic message history from a MySQL database, and features a responsive design with Bootstrap for different screen sizes. The interface includes custom avatars, time-stamped messages, and a clean, professional layout.

The project demonstrates full-stack development skills, including frontend work with HTML5, CSS3, JavaScript (jQuery), and Bootstrap, as well as backend functionality using PHP and MySQL. Users can switch between English and Spanish, type questions, and receive instant responses. The codebase includes modular files for UI, message processing, database connections, and custom scripts for message handling. Overall, JuliBot highlights interactive web applications, dynamic content handling, and bilingual user interfaces.
